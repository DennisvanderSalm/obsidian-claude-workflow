# Welcome to Your Knowledge Vault

This is your **inbox** - the landing zone for quick captures and new ideas.

## How to Use This Vault

### Folders
- **00-inbox/** - Capture first, organize later
- **01-notes/** - Processed, evergreen notes
- **02-projects/** - Active projects with defined outcomes
- **03-areas/** - Ongoing areas of responsibility (health, finance, career)
- **04-resources/** - Reference materials, bookmarks, resources
- **05-archive/** - Completed or inactive items

### With Claude Code
Run Claude Code from the repository root and ask it to:
- Help process and organize inbox items
- Find connections between notes
- Summarize topics across your vault
- Generate new content from your knowledge
