# Useful Claude Code Prompts

Copy and paste these prompts when working with your vault.

## Inbox Processing

```
Review all notes in vault/00-inbox/ and for each one:
1. Suggest which folder it belongs in
2. Recommend relevant tags
3. Identify any existing notes it should link to
```

## Finding Connections

```
Search my vault for notes related to [TOPIC].
List all relevant notes and suggest new connections I might have missed.
```

## Knowledge Gaps

```
Analyze my notes about [TOPIC] and identify:
1. What concepts are well-documented
2. What areas have gaps
3. What questions remain unanswered
```

## Weekly Review

```
Help me with my weekly review:
1. List all notes created this week
2. Summarize key themes
3. Suggest notes that need updating or connecting
4. Identify any inbox items that need processing
```

## Content Creation

```
Using my notes about [TOPIC], help me write [ARTICLE/SUMMARY/OUTLINE].
Reference specific notes where relevant.
```

## Search and Summarize

```
Find all notes tagged with #[TAG] and create a summary of the key insights.
```

## Refactor Notes

```
Review [NOTE NAME] and suggest:
1. How to make it more atomic (one idea per note)
2. What could be split into separate notes
3. What links should be added
```

## Daily Capture

```
I want to capture some thoughts about [TOPIC].
Help me structure this into a proper note with appropriate tags and links.
```
